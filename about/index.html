Hello, I am Vaishak. This is site is an effort towards honing my photography and web developments skills.

What is Project 52?

With Project 52, I hope to click post one picture every week, for the next 52 weeks. Each photograph will be based on a particular theme.

Why not Project 365?

I am sure clicking and posting one picture everyday for a years is a learning and rewarding experience, but it also requires some serious commitment. considering the time I have available for photography right now, I don't think I am ready for that kind of commitment.

What about this site?

This site is a static site generated using Jekyll. It is hosted on Github. The source for this site is available in this repository.
